# agrinho2026
# Agro Forte, Futuro Sustentável

Projeto desenvolvido para o Concurso Agrinho 2026.

## Tema

Compostagem Rural: Transformando Resíduos em Vida.

## Objetivo

Demonstrar como a compostagem pode contribuir para uma agricultura mais sustentável, reduzindo desperdícios e melhorando a qualidade do solo.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## Funcionalidades

- Apresentação do tema.
- Explicação do problema e solução.
- Calculadora de compostagem.
- Layout responsivo.
